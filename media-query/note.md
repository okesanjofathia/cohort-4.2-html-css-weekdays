### Media queries

- This is used to apply styles conditionally based on screen size, orientation or platform

### How to write media query

```css
@media (screen size in px, rem or em) {
}
```

### Illustration

---

```css
@media (max-width: 768px) {
  .container {
    display: none;
  }
}
```
